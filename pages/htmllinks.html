HTML - Links

Why?

Nice Job! We now have an HTML page with different tags. In this sections we are going to dive into hyperlinks. Links are pretty important in HTML, assuming that you obviously don't want to trap you users on one page. Links allow the user to navigate to different pages.

How?

In HTML, links are added using anchor tags. Anchor tags usually contain an href attribute, which contains the link to which you would like to direct a user. An example is: <a href="https://www.google.com" target="_blank">LINK TEXT</a>.
