HTML - Tags and Attributes

Why?

In the previous section you learned about how to structure your HTML page. Now let's have a bit more of an in-depth look into HTML tags. Tags have a name, one or more attributes and content. Names tell the browser which kind of element you're looking at. Attributes provide information on how to render the tag to the user. The content, or inner HTML normally contains the information the user will read.

How?

Let's disect the following div tag: <div class="container"><h1>Some header text</h1></div>. This tag creates a div element with an attribute class. The class element contains a value that allows you to recognize your element later on, so that you can add define how the element will look using CSS.

Fun fact

Did you notice that the div element contains another element inside of it? This is referred to as nesting. When creating HTML pages, it's common practice to nest HTML elements in order elements and even have elements inside those elements.
